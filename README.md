###	Node与MongoDB写的一个小博客。
*	前台页面包括文章列表、文章详情、评论发布与展示、登录与注册。
*	后台管理系统可以添加作者、文章、文章类型、删除文章、编辑文章、删除作者等。
*	数据库使用的MongoDB。
*	登录状态使用Cookie保存的。
*	用户密码使用Crypto加密处理。
###	安装与运行
*	本机需要安装Node与MongoDB。
*	启动本机MongoDB。
*	Clone项目到本地。
*	使用命令行进到项目根目录。
*	npm install 安装依赖。
*	npm start 启动项目。
*	浏览器地址栏输入localhost:3000或者127.0.0.1:3000即可运行项目。
